# Family Academy - Windows Installation Guide

## System Requirements
- Windows 10 or Windows 11
- 4GB RAM minimum
- 500MB free disk space
- Internet connection

## Quick Installation
1. Download `family-academy-windows-zip` from GitHub Actions
2. Extract all files to a folder (e.g., `C:\FamilyAcademy`)
3. **IMPORTANT**: Keep all files together, don't move just the .exe
4. Run `familyacademyclient.exe`

## If the app doesn't start
Install **Microsoft Visual C++ Redistributable**:
- Download from: https://aka.ms/vs/17/release/vc_redist.x64.exe
- Run the installer
- Restart your computer
- Run the app again

## Troubleshooting
- "Firebase not initialized" - This is NORMAL on Windows
- App shows blank screen - Wait 10 seconds, first load is slow
- DLL errors - Install Visual C++ Redistributable above
