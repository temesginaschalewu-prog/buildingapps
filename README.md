# Family Academy Client App

## Overview
Educational platform for students with features for learning, exams, progress tracking, and parent integration.

## Features
- User authentication and registration
- Category-based course access
- Video lessons with security features
- Study notes and practice questions
- Exam system with results tracking
- Progress tracking (Duolingo-style)
- Chatbot with AI assistance
- Payment system with proof upload
- TV device pairing
- Parent linking via Telegram
